# 🍭 Happy Corner

Welcome to **Happy Corner**, a student-led candy brand focused on delivering sweet treats with creativity, simplicity, and style.

We believe that even small ideas can turn into big projects when passion, organization, and design come together 

---

##  About Happy Corner

**Happy Corner** is a small business created in a school environment, dedicated to selling candies, snacks, and special combos at fair prices.

This project combines:
- Creativity 
- Organization  
- Modern digital presence   
- Customer-focused experience

Happy Corner uses a custom invoicing system that includes:

- Invoice number
- Order status
- Product list
- Website reference
- Telegram Orders (IN PROGRESS!!)
This helps keep orders clear, professional, and well organized.
